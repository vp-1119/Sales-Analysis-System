
# Sales & Revenue Analytics System

## Overview
A Java-based backend analytics system built using JDBC and MySQL.

## Features
- Total Revenue Calculation
- Top Selling Products
- Best Customer Analysis
- Monthly Revenue Report
- Insert New Sale (PreparedStatement)

## Tech Stack
- Java
- JDBC
- MySQL
- Maven

## Architecture
Layered Architecture:
Main → Service → Database → MySQL
